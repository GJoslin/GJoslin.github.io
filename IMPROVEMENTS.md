## Content Writing Templates

### Experience Bullet Template:
```
[Action Verb] [Specific Technology/Tool] for [Client/Project], 
[Quantifiable Result], resulting in [Business Impact]

Example:
"Architected Guidewire ClaimCenter integration for Zurich-UK, 
reducing claim processing time by 35% and integration errors by 40%, 
resulting in $500K+ annual cost savings."
```

### Project Description Template:
```markdown
## [Project Name]

**Challenge:** [What problem were you solving?]

**Solution:** [Your approach and key technologies]

**Results:**
- [Quantifiable outcome 1]
- [Quantifiable outcome 2]
- [Business impact]

**Technologies:** [List]

**Links:** [GitHub | Demo | Case Study]
```
